# Agent icon sources

These unmodified brand assets are used only to identify compatible third-party
agents in CyberCode's migration UI.

- `openclaw.png`: OpenClaw app icon from `openclaw/openclaw`.
- `claude-code.png`: Claude favicon from the official `claude.com` product site.
- `codex.png`: Codex icon distributed in the official OpenAI macOS client.
- `cursor.png`: Cursor favicon from the official `cursor.com` site.
- `hermes-agent.png`: Hermes desktop icon from `NousResearch/hermes-agent`.
- `codewhale.svg`: CodeWhale icon from `Hmbown/DeepSeek-TUI`.

All product names, logos, and trademarks belong to their respective owners.
