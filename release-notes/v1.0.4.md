# CyberCode v1.0.4

## Desktop

- Moved update availability from a floating popup to a blue update button above Settings in the left icon rail.
- The rail update button only appears when a newer desktop version is available, stays visible while downloading or restarting, and disappears when the app is already up to date.
- Removed the old global update prompt from the main app shell for a quieter update experience.
- Added Chinese, English, Japanese, and Korean labels for the new update rail states.

## Release

- Hardened the release script so version bumps do not refresh unrelated Rust dependencies in `Cargo.lock`.

## Verification

- Ran focused update and icon rail regression tests.
- Ran desktop TypeScript checks.
- Ran the desktop production web build.
