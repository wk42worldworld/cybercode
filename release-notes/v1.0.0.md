# CyberCode v1.0.0

The first release of CyberCode — a locally runnable client heavily inspired by Claude Code, with a fresh visual identity referencing Open WebUI.

## Highlights

- 🎨 **New visual identity**: cyberpunk anime branding + Open WebUI-inspired neutral grey design system (Archivo Variable font, Apple-style minimalist layout)
- 💻 **Desktop app for 4 platforms**: macOS Apple Silicon, macOS Intel, Linux x64, Windows x64
- 🤖 **Full Claude Code feature set**: Ink TUI, MCP servers, plugins, Skills system, multi-agent orchestration, Computer Use (mac/win), IM channels (Telegram/Feishu/Discord)
- 🔌 **Pluggable provider system**: works with any Anthropic-compatible API — DeepSeek, Zhipu GLM, Kimi, MiniMax, OpenRouter, LM Studio, Ollama, custom endpoints

## ⚠️ macOS users: "App is damaged" warning

The macOS app is **not** signed with an Apple Developer ID (we don't have $99/year for the Apple Developer Program). When you double-click the .dmg and try to open the app, macOS will show:

> "CyberCode.app" is damaged and can't be opened. You should move it to the Trash.

**This is a false positive.** The app is fine — Gatekeeper just refuses unsigned apps downloaded from the internet. To fix:

```bash
sudo xattr -cr /Applications/CyberCode.app
```

Enter your admin password and try opening the app again.

## Linux users

Install the `.deb` package:

```bash
sudo dpkg -i CyberCode_1.0.0_amd64.deb
sudo apt-get install -f
cybercode-desktop
```

## Windows users

Run the `.exe` installer. SmartScreen will warn — click **More info** → **Run anyway**.

See the [Installation Guide](https://github.com/wk42worldworld/cybercode/blob/main/docs/desktop/04-installation.md) for full setup details.
