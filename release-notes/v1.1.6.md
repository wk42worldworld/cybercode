# CyberCode v1.1.6

## Highlights

- Guarantees that Code Graph runtime resources are present inside every final desktop installer, not only in the build workspace.
- Adds a hard release gate that unpacks macOS ZIPs, Linux DEB packages, and Windows NSIS installers before publishing.
- Includes the native `CodeGraph` agent tool, automatic Grep/rg graph mapping, token-accounting corrections, and desktop refinements from the current release line.

## Code Graph Packaging

- Requires a non-empty `schema.sql`, `tree-sitter.wasm`, `manifest.json`, CyberCode sidecar, and at least 11 language grammar files on all four targets.
- Verifies the packaged `schema.sql` SHA-256 against the value recorded in `manifest.json`.
- Blocks artifact upload and GitHub Release creation if any packaged graph resource is missing, empty, or mismatched.
- Continues to smoke-test creation of a fresh Code Graph database before Tauri packaging.

## Code Graph Agent Integration

- Provides first-class `search`, `explore`, `impact`, `architecture`, and `status` actions alongside Grep.
- Maps broad Grep and shell `rg`/`grep` results back to owning symbols and impact relationships.
- Uses local graph fallback when the desktop MCP runtime is reconnecting and follows the global Code Graph switch.
- Keeps generated per-project graph databases outside source control and release contents.

## Distribution

- Cloud builds cover macOS Apple Silicon, macOS Intel, Linux x64, and Windows x64.
- Every platform includes a ZIP download alongside its normal DMG, DEB, or NSIS installer and updater artifacts.
- Automatic updater manifests include direct GitHub downloads and the configured mirror fallbacks.

## Verification

- Directly downloaded and inspected all four previous production archives before enabling the new release gate.
- Confirmed an 8,509-byte schema, 11 language grammars, parser runtime, matching manifest hash, and native sidecar on every target.
- Passed workflow validation, shell validation, sidecar tests, fresh-index smoke tests, and four-platform packaging checks.
