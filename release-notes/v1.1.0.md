# CyberCode v1.1.0

## Highlights

- Added smooth assistant-response streaming so uneven model network chunks appear at a stable visual cadence instead of arriving in distracting bursts.
- Reworked conversation follow-scrolling to stay responsive during long answers without repeatedly launching overlapping smooth-scroll animations.
- Preserved the original model response and final Markdown rendering while keeping the live presentation lightweight and Unicode-safe.

## Streaming Experience

- Coalesces live text updates with `requestAnimationFrame` and adapts reveal speed to the current backlog.
- Applies a bounded per-frame reveal rate so large chunks catch up quickly without freezing the interface.
- Keeps complete Unicode code points intact while streaming emoji and other multi-unit characters.
- Honors reduced-motion preferences by showing new content immediately.
- Switches back to the normal Markdown renderer when the turn completes, including inline images and document formatting.

## Conversation Scrolling

- Coalesces repeated streaming scroll requests into one animation-frame update.
- Continues following the answer only while the reader remains near the bottom.
- Cancels pending follow-scroll work when generation stops, the active conversation changes, or the message list unmounts.
- Keeps manual history reading stable while new output continues in the background.

## Distribution

- Cloud builds cover macOS Apple Silicon, macOS Intel, Linux x64, and Windows x64.
- Every platform includes a standalone ZIP download in addition to its normal DMG, DEB, or NSIS installer and updater artifacts.
- macOS builds use the configured signing and notarization pipeline; updater manifests retain the existing GitHub mirror fallbacks.
