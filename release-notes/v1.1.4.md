# CyberCode v1.1.4

## Highlights

- Fixes Code Graph projects that could remain stuck at "Preparing" without starting an index process.
- Automatically detects and rebuilds incomplete local graph databases left by an interrupted or older index run.
- Reduces automatic graph-context overhead and makes Token Optimization estimates more conservative and easier to understand.

## Code Graph Reliability

- Starts normal indexing immediately instead of waiting for active-session tool reconfiguration.
- Reattaches graph tools in the background after indexing completes, without blocking the ready state.
- Stops repeatedly restarting a failed index on every status poll and allows errored projects to be rebuilt from the UI.
- Preserves automatic indexing for newly opened and newly created projects across desktop sessions.

## Token Optimization

- Reduces the automatic Code Graph preflight budget from 1,800 to 640 tokens and limits architecture summaries to 320 tokens.
- Avoids injecting unrelated architecture context when graph search finds no matching symbols.
- Sends fewer graph nodes, relationships, and targeted source blocks while retaining the most relevant context.
- Compacts Ponytail and Caveman instructions to reduce their fixed per-request overhead.
- Uses overlap-aware combined estimates and only counts Code Graph savings when the current index is ready and non-empty.
- Shows modest positive lower estimates for longer coding tasks while retaining the short-task overhead warning.

## Distribution

- Cloud builds cover macOS Apple Silicon, macOS Intel, Linux x64, and Windows x64.
- Every platform includes a ZIP download alongside its normal DMG, DEB, or NSIS installer and updater artifacts.
- Updater manifests retain direct GitHub downloads plus the configured mirror fallbacks.

## Verification

- Passed 511 desktop tests, 31 focused Code Graph and optimization tests, and 14 sidecar tests.
- Passed desktop TypeScript checks, production compilation, native sidecar packaging, and real incomplete-index recovery checks.
