# CyberCode v1.1.5

## Highlights

- Promotes Code Graph to a native agent tool alongside Grep, so models can query symbols, architecture, dependencies, and modification impact directly.
- Fixes packaged desktop builds that could fail to create a new graph with a missing `schema.sql` error.
- Corrects current-turn and session token totals across multi-request, tool-use, streaming, and attachment-only conversations.
- Improves chat layout consistency, Code Graph visibility, provider connection tests, and anonymous usage reporting reliability.

## Code Graph

- Adds a first-class `CodeGraph` tool with `search`, `explore`, `impact`, `architecture`, and `status` actions.
- Keeps internal Code Graph MCP implementation tools out of the model-facing tool list while retaining local fallback when the runtime reconnects.
- Automatically maps broad Grep and shell `rg`/`grep` results back to owning symbols and impact relationships.
- Expands automatic graph preflight for natural-language coding tasks while respecting the global Code Graph switch immediately in both directions.
- Bundles the database schema and parser runtime in every desktop target, validates both before indexing, and smoke-tests creation of a fresh graph database.
- Excludes generated per-project `.codegraph` databases from source control and release contents.
- Updates the Chinese, English, Japanese, and Korean Code Graph guides with the native tool workflow and current context budgets.

## Token Accounting

- Deduplicates partial and final streaming usage records instead of counting the same model request more than once.
- Separates the latest API request, the complete current user turn, and the whole-session total.
- Includes tool-result continuations, side-chain records, multiple model requests, and attachment-only prompts in the correct turn.
- Keeps the compact composer value and its detailed Token panel aligned to the same current-turn definition.

## Desktop Experience

- Aligns message bubbles, tool blocks, thinking panels, task controls, and the composer to one stable chat content column.
- Improves narrow-window spacing and keeps the knowledge entry directly accessible from the sidebar.
- Adds native Code Graph labels and summaries to tool execution blocks.
- Simplifies provider connection checks to a clear success or failure result without unrelated capability text or image probing.

## Reliability And Privacy

- Uses one stable anonymous installation identity across desktop and CLI reporters, including migration from the previous local identifier.
- Adds startup retry handling for anonymous usage reporting without collecting project content, prompts, file paths, API keys, or model responses.
- Points installation documentation at the always-current GitHub Release page.

## Distribution

- Cloud builds cover macOS Apple Silicon, macOS Intel, Linux x64, and Windows x64.
- Every platform includes a ZIP download alongside its normal DMG, DEB, or NSIS installer and updater artifacts.
- Updater manifests retain direct GitHub downloads plus the configured mirror fallbacks.

## Verification

- Passed 518 desktop tests, 161 focused service and tool tests, 43 conversation integration tests, and 14 sidecar tests.
- Passed desktop TypeScript checks, production compilation, four-language documentation build, native sidecar packaging, and a real fresh Code Graph index smoke test.
