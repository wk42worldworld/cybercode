# CyberCode v1.1.3

## Highlights

- Opens the new Knowledge Space directly from the Code Graph rail button instead of routing through Token Optimization.
- Combines project graph exploration, local files, searchable knowledge sources, and source management in one focused workspace.
- Adds province-level China analytics to the private telemetry dashboard while preserving the existing anonymous, coarse-location privacy model.

## Knowledge Space

- Adds local files and folders through native pickers or drag and drop, then indexes them in the background without extra services or setup.
- Supports fast full-text search for English and Chinese content, source-scoped search, incremental reindexing, and document inspection.
- Treats common binary formats as filename and path metadata, limits oversized text indexing, skips generated dependency folders, and refuses disk-root or full-home indexing.
- Removes only CyberCode's derived index when a source is removed; original files and folders are never deleted.
- Keeps the Code Graph canvas full-height inside the expanded workspace and preserves automatic project graph behavior.

## Analytics And Privacy

- Aggregates all observed Chinese province-level regions independently from the global top-region list.
- Normalizes mainland provinces with ISO subdivision codes and includes Hong Kong, Macao, and Taiwan in the province view.
- Shows China totals, located users, users awaiting a refreshed location, a 34-region heat view, and a complete observed-province ranking.
- Continues to discard IP addresses immediately after offline country and first-level-region resolution; no city, coordinate, project, prompt, conversation, or identity data is stored.

## Distribution

- Cloud builds cover macOS Apple Silicon, macOS Intel, Linux x64, and Windows x64.
- Every platform includes a ZIP download alongside its normal DMG, DEB, or NSIS installer and updater artifacts.
- macOS packages remain signed and notarized, and updater manifests retain direct GitHub downloads plus configured mirror fallbacks.

- Passed desktop TypeScript checks, production compilation, analytics API deployment checks, responsive browser checks, and release configuration validation.
