## Highlights

- Improved busy-turn input handling in the desktop chat: messages typed while the assistant is working are now shown as separate queued rows, stay individually editable/cancellable, and are automatically sent as the next user turn when the current response finishes.
- Removed the noisy pending-input helper copy from the cached input bar, leaving only the saved message content and compact action buttons.
- Added lightweight project memory for temporary sessions, so a new temporary session can reuse relevant context from earlier temporary work without exposing the injected memory block in visible chat history or search text.
- Polished message actions with icon-only copy controls, better selection styling, and a little more top spacing for the first visible chat bubble.

## Verification

- Added regression coverage for pending input auto-send, per-row pending input actions, session search project memory indexing, and hidden project-memory context handling.
- Verified desktop tests, desktop type-check, production desktop build, and the non-desktop Bun test suite before release.
